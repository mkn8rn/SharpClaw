# Disclaimer

SharpClaw is in **alpha**. It is provided as-is, with no warranty of any kind.

## What that means

This project is built and maintained by a single developer with QA assistance from a small number of contacts. There is no dedicated test team, no formal release process, and no coverage guarantee. At this stage, undiscovered bugs are not a question of if but how many.

Things that are likely to happen:

- Features that work in one configuration break in another
- Edge cases in the permission system that nobody has hit yet
- Module interactions that were never tested together
- API surface that changes without deprecation notice
- Documentation that describes intent rather than current behavior

## What you should expect

- Breaking changes between versions
- Incomplete error handling in places
- Gaps between what the architecture promises and what the code delivers today
- Rough edges in the CLI, the Uno client, and the module APIs

## What helps

Bug reports, reproduction steps, and honest feedback. If something is broken, confusing, or just wrong, open an issue:

https://github.com/mkn8rn/SharpClaw/issues

Every report matters. This is how the project gets better.
