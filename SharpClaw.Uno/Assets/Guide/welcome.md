<![CDATA[# Welcome to SharpClaw

SharpClaw is a powerful AI agent platform with a beautiful terminal-themed interface. This guide will help you master all its features.

## What is SharpClaw?

SharpClaw combines AI chat capabilities with advanced agent orchestration, allowing you to:

- Chat with multiple AI models from different providers
- Create specialized agents with custom roles and permissions
- Organize conversations into channels and threads
- Execute tasks and jobs with full approval workflows
- Integrate bots across Telegram, Discord, WhatsApp, and more
- Run a public-facing gateway for external access

## Quick Start

1. **First Time Setup**: On your first launch, you'll create an admin account
2. **Add Providers**: Configure API keys for OpenAI, Anthropic, or other AI providers
3. **Create Agents**: Sync agents from models or create custom ones
4. **Start Chatting**: Select or create a channel to begin your conversation

## Navigation

- **Sidebar**: Lists all your channels organized by context
- **Chat Area**: View conversation history and send messages
- **Settings**: Manage providers, models, agents, roles, and more
- **User Guide** (this page): Comprehensive documentation

## Key Concepts

**Channels** are conversation spaces with specific agents and settings.

**Threads** organize related messages within a channel, each with its own history limits.

**Agents** are AI assistants powered by specific models with customizable behavior.

**Roles** define what agents and users can do via permission sets.

**Jobs** are background tasks agents can execute (shell commands, screenshots, etc.).

**Tasks** are C# automation scripts that can create agents, channels, and orchestrate workflows.
